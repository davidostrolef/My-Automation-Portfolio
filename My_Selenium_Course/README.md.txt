# Selenium & Python Automation Course - Portfolio

This repository contains a collection of automation scripts and Python logic exercises developed during my QA Automation training. Each script demonstrates different capabilities of Selenium WebDriver and Python programming.

## 📁 Project Structure & Descriptions

### 00. Python Logic Basics (`00_python_logic_basics.py`)
* **Logic Practice:** Implementation of the "7 Boom" game.
* **Functional Programming:** A "Toast Shop" calculator that uses functions, loops, and user input validation to calculate prices based on age, toppings, and drinks.

### 01. Selenium Basics & Tools (`01_selenium_basics_and_tools.py`)
* **Core Concepts:** Introduction to various Locators (ID, Name, XPath, CSS Selector).
* **Browser Interactivity:** Handling Alerts, iFrames, and multiple Browser Tabs/Windows.
* **Advanced Actions:** Demonstration of `ActionChains` and taking automated screenshots upon failures.

### 05. Table Data Automation (`05_table_data_automation.py`)
* **Data Extraction:** Iterating through web tables to extract rows and columns.
* **Data Analysis:** Finding maximum values, calculating sums (change percentages), and filtering data based on complex business logic (e.g., companies starting with "A" in specific groups).

### 06. Mortgage Calculator Flow (`06_mortgage_calculator_flow.py`)
* **Full Flow Simulation:** Automating a complete mortgage calculation process.
* **Interaction:** Using Sliders, Radio buttons, and handling Tooltips via `ActionChains`.
* **Validation:** Testing form errors and ensuring data integrity between calculation results and contact forms.

### 07. Villas Search & Filter (`07_villas_search_and_filter.py`)
* **Search Functionality:** Performing advanced searches on "Vila4U".
* **Dropdowns:** Handling `<select>` elements using the `Select` class.
* **Filtering:** Multi-selection of amenities and extracting dynamic search results.

### 08. Dynamic Reviews Scraper (`08_dynamic_reviews_scraper.py`)
* **Infinite Loading:** Handling "Load More" buttons using JavaScript execution and loops.
* **Scraping:** Extracting ratings and performing statistical analysis (average score calculation).
* **Bug Detection:** Logic to compare the declared number of reviews on the site vs. the actual scraped data.

### 09. Google Translate Automation (`09_google_translate_automation.py`)
* **Multi-language Support:** Automating translation into 5 different languages (English, Arabic, Russian, French, Latin).
* **UI Interaction:** Navigating complex dynamic menus in an RTL (Right-to-Left) environment.
* **Media Testing:** Validating and triggering the "Listen" (text-to-speech) functionality for each translation.

## 🛠 Technologies Used
* **Language:** Python 3.x
* **Library:** Selenium WebDriver
* **IDE:** PyCharm
* **Browser:** Google Chrome (ChromeDriver)
